# Uni USSD Prototype

A lightweight USSD simulation for checking student results and fee balance without internet.  
Built as a prototype to reduce portal traffic and help students during exam season.

## What it does
- **Check Results**: Enter admission number to view mock grades
- **Check Fee Balance**: View fee status instantly  
- **Help**: Basic instructions for students
- **Works offline**: Runs in any phone browser, no app needed

## How to test
1. Open `index.html` on your phone or laptop
2. Use test admission numbers:
   - `COM12345` 
   - `COM12346`

## Why this matters
- Students get results instantly without portal crashes
- Reduces admin calls during peak periods
- Can be connected to real USSD/Safaricom gateway later
